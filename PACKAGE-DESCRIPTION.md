# Package Description

Social Demand Signal Agent is an agentic social listening system that finds customer pain and buying intent, drafts human-reviewed A/B responses, and learns from tracked outcomes.

The package connects work that often lives in separate tools:

- listening for customer pain and buying intent
- qualifying and deduplicating public signals
- separating opportunity from noise, risk, and escalation
- drafting response variants from approved company claims
- requiring human review before any response is posted
- assigning experiments before outcomes exist
- tracking what happens after a response
- carrying validated learning into the next run

It is intended for:

- growth and demand generation
- community and social teams
- customer success
- founder-led sales
- product research
- company-specific social listening workflows
- Agent Skills-compatible clients

The system combines agent judgment with deterministic controls. The host agent handles context, classification, and writing. Local scripts handle ingestion, normalization, deduplication, assignment, validation, persistence, and reporting.

The package does not post responses. It keeps outbound action under human control and excludes fixture outcomes from operational learning.
