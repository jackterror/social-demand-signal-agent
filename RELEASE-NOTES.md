# Social Demand Signal Agent v0.1.0

Social Demand Signal Agent connects social listening to governed response and learning.

The first release includes:

- guided company and listening-policy onboarding
- SocialCrawl, JSON, and fixture collection paths
- local credential storage and connection testing
- deterministic normalization, deduplication, routing, and A/B assignment
- host-agent contracts for contextual classification and response writing
- human approval before any public response
- SQLite outcome storage and experiment interpretation
- profile migration, backup, import, export, and diagnostics
- local review, setup, and experiment views
- tests, evals, release audits, and Agent Skills packaging

The system does not post responses, send direct messages, or learn from fixture outcomes.
